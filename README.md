# To-do-list
To-Do List (Python Project)

This is a simple command-line To-Do List application made using Python. It helps you add, view, delete, and update tasks easily. The program uses a list to store tasks and runs using a menu-based system inside a while loop.

⸻

Features
	•	Add new tasks
	•	View all tasks
	•	Delete a task
	•	Update or edit an existing task
	•	Exit the program safely

⸻

Concepts Used
	•	Lists ([])
	•	While loop (while True)
	•	If-else conditions
	•	User input (input())
	•	Enumerate function
	•	Basic validation

⸻

Why This Project Is Useful

This project is perfect for beginners who want to practice Python and logic building. It teaches how to handle real-world tasks, how to manage data in a list, and how to interact with users through a menu.

⸻

Future Improvements (Ideas)

You can upgrade it by:
	•	Saving tasks in a file (so they don’t disappear)
	•	Adding priority to tasks
	•	Setting due dates
	•	Creating a GUI using Tkinter
	•	Making a mobile app using Kivy
